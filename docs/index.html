Agora Translator Bot

Real-time speech translation via OpenAI Realtime API and Agora Server Gateway. One speaker, one listener — conversation in two languages.

Configuration

🎙️

Speaker

Joins as host, publishes microphone audio. The bot translates your speech and publishes it back.

🎧

Listener

Joins as audience, subscribes to the bot's translated audio stream and shows captions.

You need one Speaker tab and one Listener tab — both can be on the same device.

How it works

Speaker mic → Agora RTC → Translator Bot (UID 2002) → OpenAI Realtime → Translated audio → Listener
What you'll hear
  • Speaker speaks in the source language (e.g. English).
  • The bot receives the audio, sends it to OpenAI Realtime for translation.
  • OpenAI returns translated audio (e.g. Spanish), which the bot republishes under UID 2002.
  • The listener subscribes to UID 2002 and hears the translated speech.
  • Both pages show live captions — source on the left, translation on the right.

Prerequisites

  • Token auth disabled — your Agora project must have token authentication turned off (App ID only mode).
  • Bot must be running — start the Docker container before joining. See README for instructions.
  • Same channel — all three (speaker, listener, bot) must join the same channel name.
  • Bot UID — the listener automatically subscribes to the bot's UID (default: 2002). Match --botUid if you changed it.
